# UPS/DPD 模板文件

此目录用于存放 UPS 和 DPD 的 Excel 模板文件。

## ⚠️ 重要：如何解决"找不到默认模板文件"错误

**如果你看到此错误，请选择以下任一方案：**

### 🎯 方案 1：使用自定义模板（推荐，最快捷）

1. 打开应用，进入 **UPS/DPD** 页面
2. 点击 **「模板设置」** Tab
3. 点击 **「选择自定义模板」** 按钮
4. 选择你的 Excel 模板文件（任意 .xlsx 文件都可以用于测试）
5. 开始使用！

**优势**：无需准备特定格式的模板，可以使用任何现有的 Excel 文件测试功能。

### 📁 方案 2：准备默认模板文件

将以下文件放置到本目录（`src-tauri/resources/templates/`）：
- `UPS总结单模板.xlsx`
- `DPD数据预报模板.xlsx`

**注意**：文件名必须完全一致！

---

## 所需文件

1. **UPS总结单模板.xlsx**
   - 包含以下工作表：
     - 总结单
     - 运单信息
     - 统计
     - 德国邮编
     - 子单号

2. **DPD数据预报模板.xlsx**
   - 包含以下工作表：
     - List （运单清单）
     - 总结单
     - 子单号

## 模板要求

### UPS 模板

#### 总结单工作表
- 表头应包含：Tracking Number, Packages, G.W, V.G, ZIP code, country

#### 运单信息工作表
- 表头应包含：参考号（Reference NO), 件数(PCS), 实重(Kg), 材重(Kg), 目的地(Destination), UPS主运单号(Tracking Number), 提单号（集装箱/空运）

#### 统计工作表
- 表头应包含：Destination, Package, G.W, V.W

#### 德国邮编工作表
- 表头应包含：zipcode, PCS, GW, VW, country

#### 子单号工作表
- 表头应包含：参考号（Reference NO), UPS 子单号(Tracking Number)

### DPD 模板

#### List（运单清单）工作表
- 表头应包含：Remark（箱唛 or  FBA ）, Tracking No, County, PCS, GW (kg), VW（kg), Cubic Number(CBM), post Code

#### 总结单工作表
- 第3行（索引2）：
  - F列（索引5）开始：13个德国特定邮编（4347, 6126, 14656, 21423, 36251, 39171, 44145, 47495, 56068, 59368, 67227, 75177, 90451）
  - S列（索引18）开始：其他7个国家（FR, IT, ES, NL, PL, CZ, BE）
  - Z列（索引25）：Other
  - AA列（索引26）：Total
- 第4行（索引3）：数据填充行

#### 子单号工作表
- 表头应包含：参考号 （必填）, 子单号（必填）, 主单号（必填）, 公司, 收件人, 方数

## 使用说明

1. 将模板文件放置在此目录下
2. 确保文件名与上述要求一致
3. 应用会自动使用这些模板文件进行数据处理

## 注意事项

- 模板文件必须是 .xlsx 格式
- 不要修改工作表名称和表头字段名称
- 保持模板的原有格式和样式
