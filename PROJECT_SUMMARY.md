## 运行说明

### 开发模式

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

应用将在 `http://localhost:1420` 运行

### 生产构建

```bash
# 构建应用
npm run tauri build
```

生成的可执行文件位于 `src-tauri/target/release/bundle/`
